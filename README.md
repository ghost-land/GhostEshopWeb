# GhostEshopWeb

This repo is the main website of Ghost eShop project available [here](http://ghosteshop.com).

## Development setup

```bash
# Install dependencies
$ yarn install

# Launch dev server
$ yarn dev

# Generate static files
$ yarn generate
```
